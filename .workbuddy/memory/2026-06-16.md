## repairAIJson 新增策略 9：逐行括号平衡修复
- 问题：7 个 sheet_7 的 insert 操作**全部缺外层 }**（结尾是 `"},` 而非 `"}},`），报错 "Expected double-quoted property name at position 4299 (line 22 column 5)"
- 根因：AI 在只有 3 个字段（"0","1","2"）的短操作行系统性地漏写外层 }，7 行同时缺括号导致策略 7 的组合枚举（最多4处）无法覆盖
- 修复方案：新增策略 9，按行扫描包含 `"action"` 的操作行，对括号不平衡的行在行尾补上缺失数量的 }
- 第一版 bug：`startsWith('{')` 误匹配了最外层 `{` 行，导致修成 `{}` 破坏整体结构 → 修复：增加 `includes('"action"')` 条件
- 实测：3行缺} 和 7行缺} 均成功修复
