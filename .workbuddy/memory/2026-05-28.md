## JSON 修复器（repairAIJson）
- 问题：AI 在 JSON 字符串值里写裸换行符导致 `JSON.parse` 直接炸掉
- 新增 `repairAIJson(raw)` 函数，6 级渐进修复策略：
  - 策略 0：原文直接解析
  - 策略 1：转义字符串内裸换行符（覆盖 90% 真实报错）
  - 策略 2：+ 移除 JSON 中的注释
  - 策略 3：+ 移除尾随逗号
  - 策略 4：+ 智能单引号→双引号 + 无引号key加引号 + 补齐未闭合括号
  - 策略 5：不转义换行的全套修复 fallback
  - 策略 6：提取最外层 {} 内容后全套修复
- `extractDataFromMessage` 和 `parseDataUpdate` 均已改用 `repairAIJson`
- 8 个测试用例全部通过（裸换行、尾随逗号、单引号、截断、正常JSON不被破坏、混合、注释、撇号）
